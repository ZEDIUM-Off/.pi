# Pi Norm Bundle v3

## Anchor

You must route norms deterministically instead of asking agents to remember what to read.

## Semantic orientation

You must inject broad norms at runtime and inject precise doctrine when file attributes prove it is relevant.

## Operational rules

You must install this bundle under `~/.pi`.

You must keep `~/.pi/CONTEXT.json` as the global injection entrypoint.

You must keep norm files short, imperative, and scoped by philosophy or trigger.

You must run `/ct-validate` after editing the config.

You must run `/ct-explain <path> <hook>` when injection coverage is unclear.

You must not inject `AGENTS.md` through Context Tree because Pi already loads project agent context.
